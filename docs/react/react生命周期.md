## 生命周期

> 生命周期是组件从创建到挂载页面，到更新，到页面卸载的过程

注意：类组件需要实例化，有生命周期 函数组件不需要实例化，没有生命周期

## 生命周期 - 挂载阶段

> 组件的生命周期是指组件从被创建到挂载到页面中运行起来，再到组件不用时卸载的过程，注意，只有类组件才有生命周期（类组件 实例化 函数组件 不需要实例化）

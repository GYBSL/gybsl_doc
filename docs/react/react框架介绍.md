## React特点

**1、声明式**

> 声明式的意思就是写页面UI可以跟写HTML一样，抛弃了命令式的繁琐

**2、组件化**

> 组件化可以像搭积木一样构建页面，组件化也能够实现代码复用性和可维护性 

**3、跨平台**

> react不仅仅可以构建web应用，使用 react-native 还可以构建如安卓和ios的原生应用，一套代码多个平台
